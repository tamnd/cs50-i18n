# Tuần 6 *Mở rộng*

Scalability (Khả năng mở rộng). MySQL: Số nguyên (Integers), Chuỗi (Strings), Ngày tháng (Dates), Thời gian (Times), Số thực (Real Numbers), Sai số dấu phẩy động (Floating-Point Imprecision), Độ chính xác cố định (Fixed Precision), Thay đổi bảng (Altering Tables). Stored Procedures (Thủ tục lưu trữ). PostgreSQL. Vertical Scaling (Mở rộng theo chiều dọc). Horizontal Scaling (Mở rộng theo chiều ngang). Replication (Sao chép). Read Replicas (Bản sao chỉ đọc). Sharding (Phân mảnh). Access Controls (Kiểm soát truy cập): `GRANT`, `REVOKE`. Tấn công SQL Injection. Prepared Statements (Câu lệnh chuẩn bị trước).

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, nhập
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/6/src6.zip
   ```
   
   sau đó nhấn Enter để tải xuống tệp Zip chứa mã nguồn của bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, nhập
   
   ```
   unzip src6.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, nhập
   
   ```bash
   cd src6
   ```
   
   sau đó nhấn Enter để “chuyển thư mục” vào thư mục `src6` đó.
5. Trong cửa sổ terminal, nhập, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
