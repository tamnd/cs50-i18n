## 做什么

### 准备工作

在开始本周的问题之前，你需要完成一些准备步骤。放心，接下来的几周设置会快得多！

1. 访问 [此链接](https://submit.cs50.io/invites/7caa95280ac7449ca8f939de45790f13)，使用你的 GitHub 账户登录，然后选择 **Authorize CS50**
2. 勾选表示你愿意授权课程工作人员访问你的提交内容的复选框，然后点击 **Join course**
3. 使用你的 GitHub 账户登录 [cs50.dev](https://cs50.dev/)
4. 在 codespace 的终端窗口中运行 `update50` 以确保你的 codespace 是最新的，并在提示时点击 **Rebuild now**
5. 打开 CS50 Menu 扩展，例如通过点击左侧活动栏中的 **CS50** 图标。点击 **RStudio** 以启动 RStudio。在提示时点击 **Open in Browser**。
