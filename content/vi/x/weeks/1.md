# Tuần 1 *C*

C. Mã nguồn. Mã máy. Trình biên dịch. Tính đúng đắn, Thiết kế, Phong cách. Visual Studio Code. Tô sáng cú pháp. Chuỗi thoát. Tệp tiêu đề. Thư viện. Trang hướng dẫn. Kiểu dữ liệu. Câu lệnh điều kiện. Biến. Vòng lặp. Linux. Giao diện đồ họa người dùng (GUI). Giao diện dòng lệnh (CLI). Hằng số. Chú thích. Mã giả. Toán tử. Tràn số nguyên. Sự thiếu chính xác của số thực dấu phẩy động.

Cách tải mã nguồn vào VS Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/2025/fall/lectures/1/src1.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip mã nguồn của bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src1.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src1
   ```
   
   sau đó nhấn Enter để “thay đổi thư mục” vào thư mục `src1` đó.
5. Trong cửa sổ terminal, gõ, ví dụ,
   
   ```
   make hello0
   ```
   
   sau đó nhấn Enter để biên dịch, ví dụ, `hello0.c` (mã nguồn) thành `hello0` (mã máy).
6. Trong cửa sổ terminal, gõ, ví dụ,
   
   ```
   ./hello0
   ```
   
   sau đó nhấn Enter để chạy, ví dụ, `hello0`.
