# 第 2 周 *设计*

模式（Schemas）。规范化。数据类型。存储类。类型亲和性。表约束：`PRIMARY KEY`、`FOREIGN KEY`。列约束：`CHECK`、`DEFAULT`、`NOT NULL`、`UNIQUE`。修改表：`DROP TABLE`、`ALTER TABLE`、`ADD COLUMN`、`RENAME COLUMN`、`DROP COLUMN`。Charlie。

如何将讲座源代码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/2/src2.zip
   ```
   
   然后按回车键，以便将讲座源代码的 Zip 文件下载到您的 codespace 中。
3. 在终端窗口中，输入
   
   ```
   unzip src2.zip
   ```
   
   然后按回车键来解压（即解压缩）该 Zip 文件。
4. 在终端窗口中，输入
   
   ```bash
   cd src2
   ```
   
   然后按回车键，以便“更改目录”进入 `src2` 目录（即文件夹）。
5. 在终端窗口中，例如输入
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
