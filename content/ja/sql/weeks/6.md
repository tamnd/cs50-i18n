# 第6週 *スケーリング*

スケーラビリティ。MySQL: 整数、文字列、日付、時間、実数、浮動小数点の精度不足、固定精度、テーブルの変更。ストアドプロシージャ。PostgreSQL。垂直スケーリング。水平スケーリング。レプリケーション。リードレプリカ。シャーディング。アクセス制御: `GRANT`、`REVOKE`。SQLインジェクション攻撃。プリペアドステートメント。

講義のソースコードをVisual Studio Codeにダウンロードする方法

1. [cs50.dev](https://cs50.dev/) にログインします。
2. ターミナルウィンドウで以下を入力し、
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/6/src6.zip
   ```
   
   Enterキーを押して、講義のソースコードのZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで以下を入力し、
   
   ```
   unzip src6.zip
   ```
   
   Enterキーを押して、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで以下を入力し、
   
   ```bash
   cd src6
   ```
   
   Enterキーを押して、ディレクトリ（フォルダ）をその `src6` ディレクトリに移動（「チェンジディレクトリ」）します。
5. ターミナルウィンドウで、例えば以下を入力し、
   
   ```bash
   ls
   ```
   
   Enterキーを押して、ファイルを確認します。
