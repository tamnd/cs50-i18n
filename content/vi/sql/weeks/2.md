# Tuần 2 *Thiết kế*

Lược đồ (Schemas). Chuẩn hóa (Normalizing). Kiểu dữ liệu (Data Types). Lớp lưu trữ (Storage Classes). Kiểu tương đồng (Type Affinities). Ràng buộc bảng (Table Constraints): `PRIMARY KEY`, `FOREIGN KEY`. Ràng buộc cột (Column Constraints): `CHECK`, `DEFAULT`, `NOT NULL`, `UNIQUE`. Chỉnh sửa bảng (Altering Tables): `DROP TABLE`, `ALTER TABLE`, `ADD COLUMN`, `RENAME COLUMN`, `DROP COLUMN`. Charlie.

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/2/src2.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip chứa mã nguồn bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src2.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src2
   ```
   
   sau đó nhấn Enter để "chuyển thư mục" vào thư mục `src2` đó.
5. Trong cửa sổ terminal, gõ, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
