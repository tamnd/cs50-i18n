# Tuần 0 *Truy vấn*

Bảng (Tables). Cơ sở dữ liệu (Databases). Hệ quản trị cơ sở dữ liệu (Database Management Systems). SQL. SQLite. `SELECT`. `LIMIT`. `OFFSET`. `WHERE`. Phép so sánh (Comparisons). `NOT`. `NULL`. Khớp mẫu (Pattern Matching). `LIKE`. Điều kiện phức hợp (Compound Conditions). Điều kiện khoảng (Range Conditions). Sắp xếp (Ordering). Hàm tổng hợp (Aggregate Functions). `ROUND`. `DISTINCT`.

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/0/src0.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip chứa mã nguồn bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src0.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src0
   ```
   
   sau đó nhấn Enter để "chuyển thư mục" vào thư mục `src0` đó.
5. Trong cửa sổ terminal, gõ, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
