# 第 6 周 *扩展*

可扩展性。MySQL：整数、字符串、日期、时间、实数、浮点不精确性、固定精度、修改表。存储过程。PostgreSQL。垂直扩展。水平扩展。复制。只读副本。分片。访问控制：`GRANT`、`REVOKE`。SQL 注入攻击。预处理语句。

如何将课程源码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在你的终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/6/src6.zip
   ```
   
   然后按回车键，以便将课程源码的 Zip 文件下载到你的 codespace 中。
3. 在你的终端窗口中，输入
   
   ```
   unzip src6.zip
   ```
   
   然后按回车键以解压（即减压）该 Zip 文件。
4. 在你的终端窗口中，输入
   
   ```bash
   cd src6
   ```
   
   然后按回车键，以便“更改目录”进入该 `src6` 目录（即文件夹）。
5. 在你的终端窗口中，输入（例如）
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
