# 第 5 周 *优化*

索引。`CREATE INDEX`。`EXPLAIN QUERY PLAN`。覆盖索引。B 树。部分索引。`VACUUM`。并发。事务。ACID：原子性、一致性、隔离性、持久性。`BEGIN TRANSACTION`。`COMMIT`。`ROLLBACK`。竞态条件。锁。

如何将课程源码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在你的终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/5/src5.zip
   ```
   
   然后按回车键，以便将课程源码的 Zip 文件下载到你的 codespace 中。
3. 在你的终端窗口中，输入
   
   ```
   unzip src5.zip
   ```
   
   然后按回车键以解压（即减压）该 Zip 文件。
4. 在你的终端窗口中，输入
   
   ```bash
   cd src5
   ```
   
   然后按回车键，以便“更改目录”进入该 `src5` 目录（即文件夹）。
5. 在你的终端窗口中，输入（例如）
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
