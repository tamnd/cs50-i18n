## 待解决的问题

在一个名为 `me` 的文件夹中创建一个名为 `hello.c` 的文件，用 C 语言实现一个程序，提示用户输入姓名，然后向该用户问好。例如，如果用户的名字是 Adele，你的程序应该打印 `hello, Adele\n`！

提示

- 请记住，你可以使用 `get_string` 从用户那里获取一个 `string`，它是在 `cs50.h` 中声明的。
- 请记住，你可以使用 `printf` 打印 `string`，它是在 `stdio.h` 中声明的。
- 请记住，你可以使用 `printf` 中的 `%s` 来格式化 `string`。

## 演示

## 如何开始

在终端窗口中单独执行 `cd`。你会发现终端窗口的提示符类似于：

```
$
```

接下来执行

```bash
mkdir me
```

在你的 codespace 中创建一个名为 `me` 的文件夹。

然后执行

```bash
cd me
```

进入该文件夹。你现在应该看到终端提示符为 `me/ $`。现在你可以执行

```
code hello.c
```

创建一个名为 `hello.c` 的文件，并在其中编写代码。
