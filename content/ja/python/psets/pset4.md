## すること

1. [cs50.dev](https://cs50.dev/)にログインします。これはVisual Studio Code (VS Code) のクラウドベースのバージョンであり、コースに必要なものがすべてプリインストールされた、あなた専用の「codespace」を提供します。自分のMacやPCにVS CodeやPythonをダウンロードしてインストールする必要はありません！
2. codespace curvedターミナルウィンドウで `update50` を実行して、codespaceが最新であることを確認してください。プロンプトが表示されたら、**Rebuild now** をクリックします。
3. 以下の問題をすべて提出してください。
