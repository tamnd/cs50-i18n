# Week 3: Week 3 Writing - CS50's Introduction to Databases with SQL

**Video:** https://www.youtube.com/watch?v=WXk7yDqsKxs

# Week 3 *Writing*

Create, Read, Update, Delete. `INSERT INTO`. CSVs. `.import`. `DELETE FROM`. Foreign Key Constraints. `UPDATE`. Triggers. Soft Deletions.

How to download lecture source code into Visual Studio Code

1. Log into [cs50.dev](https://cs50.dev/).
2. In your terminal window, type
   
   ```
   wget https://cdn.cs50.net/sql/2023/x/lectures/3/src3.zip
   ```
   
   followed by Enter in order to download a Zip file of lecture’s source code into your codespace.
3. In your terminal window, type
   
   ```
   unzip src3.zip
   ```
   
   followed by Enter to unzip (i.e., decompress) that Zip file.
4. In your terminal window, type
   
   ```
   cd src3
   ```
   
   followed by Enter in order to “change directory” into that `src3` directory (i.e., folder).
5. In your terminal window, type, e.g.,
   
   ```
   ls
   ```
   
   followed by Enter in order to see the files.
