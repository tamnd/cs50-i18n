# 第0週 *クエリ (Querying)*

テーブル。データベース。データベース管理システム。SQL。SQLite。`SELECT`。`LIMIT`。`OFFSET`。`WHERE`。比較。`NOT`。`NULL`。パターンマッチング。`LIKE`。複合条件。範囲条件。並べ替え。集計関数。`ROUND`。`DISTINCT`。

Visual Studio Codeに講義のソースコードをダウンロードする方法

1. [cs50.dev](https://cs50.dev/)にログインします。
2. ターミナルウィンドウで、以下を入力して
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/0/src0.zip
   ```
   
   Enterキーを押し、講義のソースコードのZipファイルをcodespaceにダウンロードします。
3. ターミナルウィンドウで、以下を入力して
   
   ```
   unzip src0.zip
   ```
   
   Enterキーを押し、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで、以下を入力して
   
   ```bash
   cd src0
   ```
   
   Enterキーを押し、その`src0`ディレクトリ（フォルダー）に「ディレクトリを変更」します。
5. ターミナルウィンドウで、例えば以下を入力して
   
   ```bash
   ls
   ```
   
   Enterキーを押し、ファイルを確認します。
