# 第 4 周 *视图*

视图。`CREATE VIEW`。用于简化的视图。用于聚合的视图。临时视图。`CREATE TEMPORARY VIEW`。公用表表达式（CTE）。用于分区的视图。用于安全的视图。软删除。

如何将课程源码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在你的终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/4/src4.zip
   ```
   
   然后按回车键，以便将课程源码的 Zip 文件下载到你的 codespace 中。
3. 在你的终端窗口中，输入
   
   ```
   unzip src4.zip
   ```
   
   然后按回车键以解压（即减压）该 Zip 文件。
4. 在你的终端窗口中，输入
   
   ```bash
   cd src4
   ```
   
   然后按回车键，以便“更改目录”进入该 `src4` 目录（即文件夹）。
5. 在你的终端窗口中，输入（例如）
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
