# 第5週 *最適化*

インデックス。`CREATE INDEX`。`EXPLAIN QUERY PLAN`。カバリングインデックス。B木 (B-Trees)。部分インデックス。`VACUUM`。並行性。トランザクション。ACID: 原子性 (Atomicity)、一貫性 (Consistency)、独立性 (Isolation)、永続性 (Durability)。`BEGIN TRANSACTION`。`COMMIT`。`ROLLBACK`。競合状態 (Race Conditions)。ロック。

講義のソースコードをVisual Studio Codeにダウンロードする方法

1. [cs50.dev](https://cs50.dev/) にログインします。
2. ターミナルウィンドウで以下を入力し、
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/5/src5.zip
   ```
   
   Enterキーを押して、講義のソースコードのZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで以下を入力し、
   
   ```
   unzip src5.zip
   ```
   
   Enterキーを押して、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで以下を入力し、
   
   ```bash
   cd src5
   ```
   
   Enterキーを押して、ディレクトリ（フォルダ）をその `src5` ディレクトリに移動（「チェンジディレクトリ」）します。
5. ターミナルウィンドウで、例えば以下を入力し、
   
   ```bash
   ls
   ```
   
   Enterキーを押して、ファイルを確認します。
