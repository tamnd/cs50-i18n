# Week 0 *Securing Accounts*

Authentication. Authorization. Usernames. Passwords. Dictionary Attacks. Brute-Force Attacks. Two-Factor Authentication (2FA). Multi-Factor Authentication: Knowledge; Possession; Inherence. One-Time Password (OTP). SIM Swapping. Keylogging. Credential Stuffing. Social Engineering. Phishing. Machine-in-the-Middle Attacks. Single Sign-On (SSO). Password Managers. Passkeys.
