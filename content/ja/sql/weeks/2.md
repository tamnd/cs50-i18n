# 第2週 *設計 (Designing)*

スキーマ。正規化。データ型。ストレージクラス。型アフィニティ。テーブル制約：`PRIMARY KEY`、`FOREIGN KEY`。カラム制約：`CHECK`、`DEFAULT`、`NOT NULL`、`UNIQUE`。テーブルの変更：`DROP TABLE`、`ALTER TABLE`、`ADD COLUMN`、`RENAME COLUMN`、`DROP COLUMN`。Charlie。

Visual Studio Codeに講義のソースコードをダウンロードする方法

1. [cs50.dev](https://cs50.dev/)にログインします。
2. ターミナルウィンドウで、以下を入力して
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/2/src2.zip
   ```
   
   Enterキーを押し、講義のソースコードのZipファイルをcodespaceにダウンロードします。
3. ターミナルウィンドウで、以下を入力して
   
   ```
   unzip src2.zip
   ```
   
   Enterキーを押し、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで、以下を入力して
   
   ```bash
   cd src2
   ```
   
   Enterキーを押し、その`src2`ディレクトリ（フォルダー）に「ディレクトリを変更」します。
5. ターミナルウィンドウで、例えば以下を入力して
   
   ```bash
   ls
   ```
   
   Enterキーを押し、ファイルを確認します。
