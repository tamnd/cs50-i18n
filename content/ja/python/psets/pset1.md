## やること

1. [cs50.dev](https://cs50.dev/)にログインします。これはVisual Studio Code (VS Code) のクラウドベース版で、コースに必要なものがすべてプリインストールされた自分専用の「codespace」が提供されます。自分のMacやPCにVS CodeやPythonをダウンロードしてインストールする必要はありません！
2. codespaceのターミナルウィンドウで`update50`を実行して、codespaceが最新であることを確認してください。プロンプトが表示された場合は、**Rebuild now**をクリックしてください。
3. 以下のすべての問題を提出してください：
   
   - [Deep Thought](deep/)
   - [Home Federal Savings Bank](bank/)
   - [File Extensions](extensions/)
   - [Math Interpreter](interpreter/)
   - [Meal Time](meal/)
