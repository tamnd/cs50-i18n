# 第2週 *配列*

プリプロセス。コンパイル。アセンブル。リンク。デバッグ。配列。文字列。コマンドライン引数。暗号。

Visual Studio Codeにソースコードをダウンロードする方法

1. [cs50.dev](https://cs50.dev/)にログインします。
2. ターミナルウィンドウで、以下を入力します。
   
   ```python
   wget https://cdn.cs50.net/2025/fall/lectures/2/src2.zip
   ```
   
   入力後、Enterを押して、講義のソースコードが含まれるZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで、以下を入力します。
   
   ```
   unzip src2.zip
   ```
   
   入力後、Enterを押して、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで、以下を入力します。
   
   ```bash
   cd src2
   ```
   
   入力後、Enterを押して、その`src2`ディレクトリ（フォルダ）に「ディレクトリを移動（change directory）」します。
5. ターミナルウィンドウで、例として以下のように入力します。
   
   ```
   make hello0
   ```
   
   入力後、Enterを押して、例として`hello0.c`（ソースコード）を`hello0`（マシンコード）にコンパイルします。
6. ターミナルウィンドウで、例として以下のように入力します。
   
   ```
   ./hello0
   ```
   
   入力後、Enterを押して、例として`hello0`を実行します。
