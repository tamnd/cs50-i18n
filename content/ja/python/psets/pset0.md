## やること

1. [このフォーム](https://forms.cs50.io/27a35e60-490a-4e42-8574-6b55154ae760)を提出してください。
2. [このリンク](https://submit.cs50.io/invites/579861a2825849178cd11315c055bc49)にアクセスし、GitHubアカウントでログインして、**Authorize CS50**をクリックしてください。
3. コーススタッフに提出物へのアクセス権限を付与することを示すボックスにチェックを入れ、**Join course**をクリックしてください。
4. [cs50.dev](https://cs50.dev/)にログインします。これはVisual Studio Code (VS Code) のクラウドベース版で、コースに必要なものがすべてプリインストールされた自分専用の「codespace」が提供されます。自分のMacやPCにVS CodeやPythonをダウンロードしてインストールする必要はありません！まだ使い慣れていない場合は、概要を把握するために[Visual Studio Code for CS50 ショート](../../shorts/visual_studio_code_for_cs50/)を必ず視聴してください。
5. codespaceがロードされたら、デフォルトで開いている可能性のある**Welcome**タブをすべて閉じてください。
6. codespaceのターミナルウィンドウで`update50`を実行して、codespaceが最新であることを確認してください。プロンプトが表示された場合は、**Rebuild now**をクリックしてください。
7. 以下のすべての問題を提出してください：
   
   - [Indoor Voice](indoor/)
   - [Playback Speed](playback/)
   - [Making Faces](faces/)
   - [Einstein](einstein/)
   - [Tip Calculator](tip/)
