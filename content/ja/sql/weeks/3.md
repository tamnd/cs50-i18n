# 第3週 *書き込み*

作成 (Create)、読み取り (Read)、更新 (Update)、削除 (Delete)。`INSERT INTO`。CSV。`.import`。`DELETE FROM`。外部キー制約。`UPDATE`。トリガー。論理削除 (Soft Deletions)。

講義のソースコードをVisual Studio Codeにダウンロードする方法

1. [cs50.dev](https://cs50.dev/) にログインします。
2. ターミナルウィンドウで以下を入力し、
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/3/src3.zip
   ```
   
   Enterキーを押して、講義のソースコードのZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで以下を入力し、
   
   ```
   unzip src3.zip
   ```
   
   Enterキーを押して、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで以下を入力し、
   
   ```bash
   cd src3
   ```
   
   Enterキーを押して、ディレクトリ（フォルダ）をその `src3` ディレクトリに移動（「チェンジディレクトリ」）します。
5. ターミナルウィンドウで、例えば以下を入力し、
   
   ```bash
   ls
   ```
   
   Enterキーを押して、ファイルを確認します。
