# Tuần 1 *Quan hệ*

Cơ sở dữ liệu quan hệ (Relational Databases). Các mối quan hệ (Relationships): Một-một (One-to-one), Một-nhiều (One-to-many), Nhiều-nhiều (Many-to-many). Sơ đồ quan hệ thực thể (Entity Relationship Diagrams). Khóa (Keys): Khóa chính (Primary Keys), Khóa ngoại (Foreign Keys). Truy vấn con (Subqueries). `IN`. Liên kết (Joins): `INNER JOIN`, Outer Joins, `LEFT JOIN`, `RIGHT JOIN`, `FULL JOIN`, `NATURAL JOIN`. Tập hợp (Sets): `INTERSECT`, `UNION`, `EXCEPT`. Nhóm (Groups): `GROUP BY`, `HAVING`.

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/1/src1.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip chứa mã nguồn bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src1.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src1
   ```
   
   sau đó nhấn Enter để "chuyển thư mục" vào thư mục `src1` đó.
5. Trong cửa sổ terminal, gõ, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
