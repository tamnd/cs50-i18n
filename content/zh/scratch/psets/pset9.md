## 任务内容

访问 [scratch.mit.edu](https://scratch.mit.edu/) 并登录你的账户。点击 “Create” (创建) 按钮新建一个项目。

你的作业是创建一个新的 Scratch 项目，应用你在课程中学到的所有理念。你的项目应当包含以下**所有**内容：

- 至少三个不同的角色。
- 至少总共三个背景。
- 至少一个广播。
- 至少一个克隆。
- 至少一个自制积木。
- 至少一个变量。
- 至少一个循环。
- 至少一个条件判断。
- 至少一个事件。

如果缺少上述九项要求中的**任何一项**，你的项目都无法通过，请确保它们全部存在！

项目完成后，最后一次选择 **File (文件) > Save now (立即保存)**。然后选择 **File (文件) > Save to your computer (保存到电脑)** 并保留该文件（扩展名为 `.sb3`），以便提交。如果电脑提示 **Open** (打开) 或 **Save** (保存) 文件，请务必选择 **Save** (保存)。

## 如何提交

与课程中的大多数其他题目不同，你必须在 `check50` 中获得 100% 的分数才能通过。这是课程通常期望 70% 分数即为及格的一个例外。请务必注意！

此步骤假设你已经将 Scratch 项目下载为以 `.sb3` 结尾的文件。此外，此步骤还假设你已经按照上述表格注册了 GitHub 账户。

1. 访问[此链接](https://submit.cs50.io/invites/a6bd992871134b9da37583413652669d)，使用你的 GitHub 账户登录，然后点击 **Authorize cs50**。
2. 勾选表示你愿意授权课程工作人员访问你的提交内容的复选框，然后点击 **Join course**。
3. 访问 [submit.cs50.io/upload/cs50/problems/2024/scratch/build](http://submit.cs50.io/upload/cs50/problems/2024/scratch/build)。
4. 点击 “Choose File” (选择文件) 并选择你的 `.sb3` 文件。点击 **Submit** (提交)。

如果你遇到错误 “No files in this directory are expected by cs50/problems/2024/scratch/build”，请确保你的 Scratch 项目文件名确实以 `.sb3` 结尾！

就这样！提交上传后，你应该会被重定向到提交页面。点击提交链接，然后点击 **check50** 链接，查看你的项目满足了哪些要求。欢迎在截止日期前多次重新提交！请注意，如果你上传的文件大于 10MB（对于 Scratch 项目来说相当大），`check50` 可能会难以处理。最好确保你的文件小于该限制。

要查看你在课程中的当前进度，请访问课程成绩单 [cs50.me/cs50s](https://cs50.me/cs50s)！
