## Cần làm gì

1. Đăng nhập vào [cs50.dev](https://cs50.dev/), đây là phiên bản dựa trên đám mây của Visual Studio Code (VS Code) cung cấp cho bạn một “codespace” riêng với mọi thứ bạn cần cho khóa học đã được cài đặt sẵn. Không cần tải xuống và cài đặt VS Code hoặc Python trên Mac hoặc PC của riêng bạn!
2. Chạy lệnh `update50` trong cửa sổ terminal của codespace để đảm bảo rằng codespace của bạn đã được cập nhật; nếu được thông báo, hãy nhấp vào **Rebuild now**.
3. Nộp tất cả các bài tập dưới đây:
   
   - [Các dòng mã](lines/)
   - [Pizza Py](pizza/)
   - [Scourgify](scourgify/)
   - [Áo phông CS50](shirt/)
