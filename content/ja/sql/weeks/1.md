# 第1週 *関連付け (Relating)*

リレーショナルデータベース。関係：1対1、1対多、多対多。実体関連図（ER図）。キー：主キー、外部キー。サブクエリ。`IN`。結合：`INNER JOIN`、外部結合、`LEFT JOIN`、`RIGHT JOIN`、`FULL JOIN`、`NATURAL JOIN`。集合：`INTERSECT`、`UNION`、`EXCEPT`。グループ化：`GROUP BY`、`HAVING`。

Visual Studio Codeに講義のソースコードをダウンロードする方法

1. [cs50.dev](https://cs50.dev/)にログインします。
2. ターミナルウィンドウで、以下を入力して
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/1/src1.zip
   ```
   
   Enterキーを押し、講義のソースコードのZipファイルをcodespaceにダウンロードします。
3. ターミナルウィンドウで、以下を入力して
   
   ```
   unzip src1.zip
   ```
   
   Enterキーを押し、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで、以下を入力して
   
   ```bash
   cd src1
   ```
   
   Enterキーを押し、その`src1`ディレクトリ（フォルダー）に「ディレクトリを変更」します。
5. ターミナルウィンドウで、例えば以下を入力して
   
   ```bash
   ls
   ```
   
   Enterキーを押し、ファイルを確認します。
