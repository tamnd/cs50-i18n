# 第4週 *ビュー*

ビュー。`CREATE VIEW`。簡素化のためのビュー。集計のためのビュー。一時ビュー。`CREATE TEMPORARY VIEW`。共通テーブル式 (Common Table Expressions)。パーティショニングのためのビュー。セキュリティのためのビュー。論理削除 (Soft Deletions)。

講義のソースコードをVisual Studio Codeにダウンロードする方法

1. [cs50.dev](https://cs50.dev/) にログインします。
2. ターミナルウィンドウで以下を入力し、
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/4/src4.zip
   ```
   
   Enterキーを押して、講義のソースコードのZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで以下を入力し、
   
   ```
   unzip src4.zip
   ```
   
   Enterキーを押して、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで以下を入力し、
   
   ```bash
   cd src4
   ```
   
   Enterキーを押して、ディレクトリ（フォルダ）をその `src4` ディレクトリに移動（「チェンジディレクトリ」）します。
5. ターミナルウィンドウで、例えば以下を入力し、
   
   ```bash
   ls
   ```
   
   Enterキーを押して、ファイルを確認します。
