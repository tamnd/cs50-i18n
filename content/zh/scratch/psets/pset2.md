## 任务清单

访问 [scratch.mit.edu](https://scratch.mit.edu/) 并登录你的账号。点击 “Create”（创造）按钮创建一个新项目。

你的作业是创建一个 Scratch 动画。你的项目应该包括：

- 至少一个 **Motion**（运动）积木。
- 至少一个 **Looks**（外观）积木。
- 至少一个 **Sound**（声音）积木。

完成项目后，最后一次选择 **File &gt; Save now**（文件 &gt; 立即保存）。然后选择 **File &gt; Save to your computer**（文件 &gt; 保存到电脑），并保留该文件（文件扩展名为 `.sb3`），以便提交。如果电脑提示你 **Open**（打开）还是 **Save**（保存）文件，请务必选择 **Save**（保存）。

## 如何提交

这一步假设你已经将 Scratch 项目下载为以 `.sb3` 结尾的文件。同时，这一步也假设你已经按照上述表格注册了 GitHub 账号。

1. 访问[此链接](https://submit.cs50.io/invites/a6bd992871134b9da37583413652669d)，使用你的 GitHub 账号登录，然后点击 **Authorize cs50**。
2. 勾选表示你愿意授权课程工作人员访问你的提交作品的复选框，然后点击 **Join course**。
3. 前往 [submit.cs50.io/upload/cs50/problems/2024/scratch/functions](http://submit.cs50.io/upload/cs50/problems/2024/scratch/functions)。
4. 点击 “Choose File”（选择文件）并选择你的 `.sb3` 文件。点击 **Submit**（提交）。

如果你遇到 “No files in this directory are expected by cs50/problems/2024/scratch/functions” 的错误，请确保你的 Scratch 项目文件名确实以 `.sb3` 结尾！

大功告成！提交上传后，你应该会被重定向到提交页面。点击提交链接，然后点击 **check50** 链接，查看你的项目符合哪些要求。欢迎你在截止日期前多次重新提交！请注意，如果你上传的文件大小超过 10MB（对于 Scratch 项目来说相当大），`check50` 可能会难以处理。最好确保你的文件小于该限制。

要查看你在课程中的当前进度，请访问课程成绩单：[cs50.me/cs50s](https://cs50.me/cs50s)！
