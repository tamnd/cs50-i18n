# 第 0 周 *保护账户安全*

身份验证。授权。用户名。密码。字典攻击。暴力破解攻击。双重身份验证 (2FA)。多重身份验证：知识；持有；固有特征。一次性密码 (OTP)。SIM 卡交换。键盘记录。撞库攻击。社会工程学。钓鱼攻击。中间人攻击。单点登录 (SSO)。密码管理器。通行密钥。
