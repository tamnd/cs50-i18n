# Tuần 5 *Tối ưu hóa*

Indexes (Chỉ mục). `CREATE INDEX`. `EXPLAIN QUERY PLAN`. Covering Indexes (Chỉ mục bao phủ). B-Trees. Partial Indexes (Chỉ mục một phần). `VACUUM`. Concurrency (Tính đồng thời). Transactions (Giao dịch). ACID: Atomicity (Tính nguyên tử), Consistency (Tính nhất quán), Isolation (Tính cô lập), Durability (Tính bền vững). `BEGIN TRANSACTION`. `COMMIT`. `ROLLBACK`. Race Conditions (Tranh chấp tài nguyên). Locks (Khóa).

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, nhập
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/5/src5.zip
   ```
   
   sau đó nhấn Enter để tải xuống tệp Zip chứa mã nguồn của bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, nhập
   
   ```
   unzip src5.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, nhập
   
   ```bash
   cd src5
   ```
   
   sau đó nhấn Enter để “chuyển thư mục” vào thư mục `src5` đó.
5. Trong cửa sổ terminal, nhập, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
