# 第1週 *C*

C。ソースコード。マシンコード。コンパイラ。正確性、設計、スタイル。Visual Studio Code。構文ハイライト。エスケープシーケンス。ヘッダーファイル。ライブラリ。マニュアルページ（manページ）。型。条件分岐。変数。ループ。Linux。グラフィカルユーザインタフェース (GUI)。コマンドラインインタフェース (CLI)。定数。コメント。擬似コード。演算子。整数オーバーフロー。浮動小数点精度の問題。

VS Codeにソースコードをダウンロードする方法

1. [cs50.dev](https://cs50.dev/) にログインします。
2. ターミナルウィンドウで、以下のコマンドを入力します。
   
   ```python
   wget https://cdn.cs50.net/2025/fall/lectures/1/src1.zip
   ```
   
   続けて Enter を押し、講義のソースコードのZipファイルをコードスペースにダウンロードします。
3. ターミナルウィンドウで、以下のコマンドを入力します。
   
   ```
   unzip src1.zip
   ```
   
   続けて Enter を押し、そのZipファイルを解凍（展開）します。
4. ターミナルウィンドウで、以下のコマンドを入力します。
   
   ```bash
   cd src1
   ```
   
   続けて Enter を押し、その `src1` ディレクトリ（フォルダ）に「ディレクトリを変更（移動）」します。
5. ターミナルウィンドウで、例えば以下のように入力します。
   
   ```
   make hello0
   ```
   
   続けて Enter を押し、例えば `hello0.c`（ソースコード）を `hello0`（マシンコード）にコンパイルします。
6. ターミナルウィンドウで、例えば以下のように入力します。
   
   ```
   ./hello0
   ```
   
   続けて Enter を押し、例えば `hello0` を実行します。
