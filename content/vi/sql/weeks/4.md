# Tuần 4 *Khung nhìn*

Khung nhìn (Views). `CREATE VIEW`. Khung nhìn để đơn giản hóa (Views for Simplifying). Khung nhìn để tổng hợp (Views for Aggregating). Khung nhìn tạm thời (Temporary Views). `CREATE TEMPORARY VIEW`. Biểu thức bảng chung (Common Table Expressions). Khung nhìn để phân vùng (Views for Partitioning). Khung nhìn để bảo mật (Views for Securing). Xóa tạm thời (Soft Deletions).

Cách tải mã nguồn bài giảng vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/4/src4.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip chứa mã nguồn bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src4.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src4
   ```
   
   sau đó nhấn Enter để "chuyển thư mục" vào thư mục `src4` đó.
5. Trong cửa sổ terminal, gõ, ví dụ:
   
   ```bash
   ls
   ```
   
   sau đó nhấn Enter để xem các tệp tin.
