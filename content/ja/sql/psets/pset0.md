## やるべきこと

### 準備

今週の問題に取り掛かる前に、いくつかの準備ステップが必要です。来週以降のセットアップはもっと早く終わりますので、ご安心ください！

1. [このリンク](https://submit.cs50.io/invites/51235e610e9846fd853379c4a44e7e9c)にアクセスし、GitHub アカウントでログインして、**Authorize CS50** を選択します。
2. コーススタッフがあなたの提出物にアクセスすることを許可するチェックボックスをオンにし、**Join course** をクリックします。
3. GitHub アカウントを使用して [cs50.dev](https://cs50.dev/) にログインします。
4. コードスペースが最新であることを確認するために、コードスペースのターミナルウィンドウで `update50` を実行し、プロンプトが表示されたら **Rebuild now** をクリックします。
