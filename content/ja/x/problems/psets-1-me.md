## 解くべき問題

`me` というフォルダ内の `hello.c` というファイルに、ユーザーに名前を尋ね、そのユーザーに挨拶する C 言語のプログラムを実装してください。例えば、ユーザーの名前が Adele の場合、プログラムは `hello, Adele\n` と出力する必要があります！

ヒント

- `cs50.h` で宣言されている `get_string` を使ってユーザーから `string` を取得できることを思い出してください。
- `stdio.h` で宣言されている `printf` を使って `string` を出力できることを思い出してください。
- `printf` で `%s` を使って `string` をフォーマットできることを思い出してください。

## デモ

## 始め方

ターミナルウィンドウで `cd` だけを実行してください。ターミナルウィンドウのプロンプトが以下のようになっているはずです。

```
$
```

次に以下を実行して

```bash
mkdir me
```

codespace に `me` というフォルダを作成します。

次に以下を実行して

```bash
cd me
```

そのフォルダにディレクトリを移動します。ターミナルのプロンプトが `me/ $` と表示されるはずです。これで、以下を実行して

```
code hello.c
```

コードを記述するための `hello.c` というファイルを作成できます。
