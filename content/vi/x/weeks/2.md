# Tuần 2 *Mảng*

Tiền xử lý. Biên dịch. Hợp dịch. Liên kết. Gỡ lỗi. Mảng. Chuỗi. Đối số dòng lệnh. Mật mã học.

Cách tải mã nguồn vào Visual Studio Code

1. Đăng nhập vào [cs50.dev](https://cs50.dev/).
2. Trong cửa sổ terminal, gõ
   
   ```python
   wget https://cdn.cs50.net/2025/fall/lectures/2/src2.zip
   ```
   
   sau đó nhấn Enter để tải tệp Zip mã nguồn của bài giảng vào codespace của bạn.
3. Trong cửa sổ terminal, gõ
   
   ```
   unzip src2.zip
   ```
   
   sau đó nhấn Enter để giải nén tệp Zip đó.
4. Trong cửa sổ terminal, gõ
   
   ```bash
   cd src2
   ```
   
   sau đó nhấn Enter để “thay đổi thư mục” vào thư mục `src2` đó.
5. Trong cửa sổ terminal, gõ, ví dụ,
   
   ```
   make hello0
   ```
   
   sau đó nhấn Enter để biên dịch, ví dụ, `hello0.c` (mã nguồn) thành `hello0` (mã máy).
6. Trong cửa sổ terminal, gõ, ví dụ,
   
   ```
   ./hello0
   ```
   
   sau đó nhấn Enter để chạy, ví dụ, `hello0`.
