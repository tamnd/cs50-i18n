## 做什么

1. 使用你的 GitHub 账户登录 [cs50.dev](https://cs50.dev)，**请使用 Chrome、Edge 或 Safari**，不要使用 Firefox
2. 在 codespace 的终端窗口运行 `update50` 以确保你的 codespace 是最新的，当出现提示时，点击 **Rebuild now**
3. 提交 [Scrabble](scrabble/)
4. 提交 [Readability](readability/)
5. 提交以下之一：
   
   - 如果觉得不太自信，提交 [Caesar](caesar/)
   - 如果觉得比较自信，提交 [Substitution](substitution/)

如果你同时提交了 Caesar 和 Substitution，我们将记录两个分数中的较高者。

## 建议

- 通过 [第 2 周](../../weeks/2/) 的示例尝试 David 在课堂上的任何程序。
- 要查看 C 语言函数的联机帮助页（manual pages），请访问 [manual.cs50.io](https://manual.cs50.io/)。
- 如果在使用 `make` 编译代码时看到任何错误，请首先专注于修复看到的第一个错误，并根据需要向上滚动。如果不确定错误的含义，请尝试向 `help50` 寻求帮助。例如，如果尝试编译 `readability` 时，
  
  ```
  make readability
  ```
  
  产生错误，请尝试运行
  
  ```
  help50 make readability
  ```
  
  代替！
