# 第 0 周 *查询*

表。数据库。数据库管理系统。SQL。SQLite。`SELECT`。`LIMIT`。`OFFSET`。`WHERE`。比较。`NOT`。`NULL`。模式匹配。`LIKE`。复合条件。范围条件。排序。聚合函数。`ROUND`。`DISTINCT`。

如何将讲座源代码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/0/src0.zip
   ```
   
   然后按回车键，以便将讲座源代码的 Zip 文件下载到您的 codespace 中。
3. 在终端窗口中，输入
   
   ```
   unzip src0.zip
   ```
   
   然后按回车键来解压（即解压缩）该 Zip 文件。
4. 在终端窗口中，输入
   
   ```bash
   cd src0
   ```
   
   然后按回车键，以便“更改目录”进入 `src0` 目录（即文件夹）。
5. 在终端窗口中，例如输入
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
