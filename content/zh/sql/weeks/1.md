# 第 1 周 *关联*

关系型数据库。关系：一对一、一对多、多对多。实体关系图。键：主键、外键。子查询。`IN`。连接：`INNER JOIN`、外连接、`LEFT JOIN`、`RIGHT JOIN`、`FULL JOIN`、`NATURAL JOIN`。集合：`INTERSECT`、`UNION`、`EXCEPT`。分组：`GROUP BY`、`HAVING`。

如何将讲座源代码下载到 Visual Studio Code

1. 登录 [cs50.dev](https://cs50.dev/)。
2. 在终端窗口中，输入
   
   ```python
   wget https://cdn.cs50.net/sql/2023/x/lectures/1/src1.zip
   ```
   
   然后按回车键，以便将讲座源代码的 Zip 文件下载到您的 codespace 中。
3. 在终端窗口中，输入
   
   ```
   unzip src1.zip
   ```
   
   然后按回车键来解压（即解压缩）该 Zip 文件。
4. 在终端窗口中，输入
   
   ```bash
   cd src1
   ```
   
   然后按回车键，以便“更改目录”进入 `src1` 目录（即文件夹）。
5. 在终端窗口中，例如输入
   
   ```bash
   ls
   ```
   
   然后按回车键以查看文件。
