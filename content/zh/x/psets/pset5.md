## 该做什么

1. 使用你的 GitHub 账号登录 [cs50.dev](https://cs50.dev)
2. 在你的 codespace 终端窗口中运行 `update50` 以确保你的 codespace 是最新的，并在提示时点击 **Rebuild now**
3. 提交 [Inheritance](inheritance/)
4. 提交 [Speller](speller/)

## 建议

- 尝试运行 [第 5 周](../../weeks/5/) 中 David 的任何程序。
- 如果你在使用 `make` 编译代码时看到任何错误，请首先专注于修复你看到的第一个错误，根据需要向上滚动。如果不确定它的含义，请尝试向 `help50` 寻求帮助。例如，如果尝试编译 `speller` 并且
  
  ```
  make speller
  ```
  
  产生了错误，请尝试运行
  
  ```
  help50 make speller
  ```
  
  作为替代！
